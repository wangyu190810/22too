# 定增规则“随行就市”带来的风险和机会

我最近关注的几个股票，定增规则都变了，定增都搞了一年了，还没定增成功
弘信电子，从定增价格14块，成了"随行就市"，按照发行当日的百分之80价格定增，弘信电子的定增搞了一年半了，一直没搞定，改了规则应该很快就定增好了。
汇纳科技，主控人变更，当时的老板，大概是20多买的，然后我以为早就完事了，没想到还没批准，前几天改成了”随行就市“。

# 定增实际上解决的问题
1. 上市公司资金短缺，借款太多，负债利息太多，降低财务费用
2. 需要投资新的项目，公司没钱，但是又不想去借银行的钱，找人增发股票，直接变成无本投资，只要投资人愿意出钱就行。
3. 控制权变更，需要增发股票，用来稀释老古董，引入新的股东，这个就比较复杂了，不做单独分析。

# 以前的定增
1. 上市公司，先找好投资人，然后商量一个价格，打压一下股价，然后按照最近20日的股价，然后定增，新股东直接无风险投资，然后锁定期结束，直接卖出完事。
2. 交易所，只要不是太过分，反正你能找来投资人，那说明你牛，我给你批了就行。
3. 散户是最受伤的一群人，定增开始前股价暴跌，然后被迫减仓，定增之后，加仓又来不及，成本被抬高，最终收益有限。

# 真实的变化带来的利益变化
1. 同样的钱，定增公告之后，公司的可能买的股份更少了，投资人的利益降低，毕竟以前是直接赚百分之20，现在也就少赚一点，而不是产生损失，
2. 定增消息公布之后，散户的机会，消息公布，基本散户就可以跟着操作了，毕竟切实增强了上市公司的实力，并且降低了财务费用，实质上的利好。
3. 定增真实的变成了解决问题，增强公司实力，上市公司更需要确保定增人的顺利买单，需要更大的饼和更好的未来，才能确保定增落地。

以前看定增，就是觉得定增就是上市公式和定增方合谋，对散户来说算不上实质性利好
现在看定增，目标转向解决问题，定增价格按照随行就市，散户可以直接跟进。


*整体而言，我们可以继续关注后续有定增的公司，从而验证这个结论。*



定增规则“随行就市”：我跟踪的几个案例，暴露了机会和风险

我最近跟踪的几个股票，定增规则悄悄变了。

弘信电子，定增搞了一年半，原来锁定的价格是14块，后来改成了“随行就市”——按发行期首日前20日均价的八折定价。改了规则后，事情明显在加速推进。

汇纳科技，实际控制人变更，新老板大概20多块买的，我以为这事早该定了，结果一直没批。前阵子也改成了“随行就市”。

这些案例引发我一个思考：定增规则的“随行就市”，到底改变了什么？

定增真正在解决什么问题

抛开概念，定增就是上市公司用股权换钱，通常有三个用途：

公司缺钱，借款太多利息太重，需要降低财务费用

有新项目要投，不想借银行，找投资人直接出钱

控制权变更，通过增发稀释老股东、引入新股东（这个比较复杂，先不谈）

以前的定增：更像一场合谋

老规矩下，上市公司先找好投资人，商量一个价格。然后在股价低迷时开董事会，把定价基准日锁在那个低点，按20日均价打八折。

投资人拿到的是明面上的折扣，锁定期结束卖出走人。
监管看你情我愿，只要不太过分，一般放行。
散户呢？定增前股价被压低，被迫减仓；定增后想加仓，成本已经上去了。整个过程，散户是最被动的一群人。

现在的变化：利益链条被打散了

规则改成“随行就市”后，同一条利益链上的人，处境完全不同了：

投资人： 同样的钱，能买的股份可能变多了（因为定价跟着市场走），但安全垫变薄了。以前是坐享20%的折扣，现在赚的是市场定价的钱，少赚但不是亏。

上市公司： 定增真正变成了“解决问题”的工具。你必须拿出靠谱的项目、清晰的未来，才能让投资人在市场化定价下愿意买单。画饼不够大、不够真，定增就发不出去。

散户： 这是最大的变化。过去定增公告对散户不一定是好事，因为你不在那个“局”里。现在呢？定增消息出来，散户可以跟着分析了——公司实力确实增强了，财务费用确实降低了，这是实质利好，而且你跟投资人站在了更接近的起跑线上。

一个可以验证的结论

以前我看定增，觉得就是大股东和定增方的合谋，散户最好躲远点。
现在看定增，如果一家公司从“锁价”改为“随行就市”，我会认真看一眼：它是真的需要钱、真的在做事。而且这件事，散户是能跟的。
